# QQ窥屏检测

#### 介绍

QQ是一款热门的即时通讯软件。随着时间推移，他的功能和玩法被用户一一发现。
有些用户不说话，只看群聊消息。对于这些用户，我们不知道他是否在看着群的内容。而QQ窥屏检测可以简单、方便的检测出他们的存在。

#### 使用说明

程序会检测GET参数id，如果没有则直接退出。当程序被请求时，会在./kpData/文件夹下生成id.txt的文件，包含请求时间、用户IP、IP属地和用户UA。每条数据用回车分隔，每组数据用两个回车分隔。
